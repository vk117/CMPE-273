Instructions for running the applications:

For Calculator :-

In first console-

1. npm install
2. npm start

In second console-

1. node Server.js

 Go to localhost:3000/ to access the calculator

##################################################################################

For Freelancer web app:-

In first console-

1. npm install
2. cd react
3. npm start

In second console-

1. cd node
2. node app.js

Go to localhost:3000/ to access the freelancer web app/
